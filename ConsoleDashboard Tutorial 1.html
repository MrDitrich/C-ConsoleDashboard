Start with using statements

Import required namespaces

At the very top of your file, type:

üí° Explanation:

Create the Program class

Define the main structure

Every C# program needs a class to hold the code. Under your using statements:

class Program { static void Main() { // Your program will go here } }

üí° Explanation:

  • ‚Ä¢ class Program = container for all your code.
  • ‚Ä¢ static void Main() = entry point; the computer always starts here.
3

Declare your variables

Set up storage for data

Inside Main(), at the very top, type:

string userName; // store the user's name int mainChoice; // store menu choices (for future tutorials) int subChoice; // store sub-menu choices (for future tutorials)

üí° Why at the top?

  • ‚Ä¢ Makes your code easier to read.
  • ‚Ä¢ You know all the variables your program will use.
  • ‚Ä¢ Avoids errors if you accidentally declare the same variable twice later.

Display a welcome banner

Create visual appeal

Type this next:

Console.ForegroundColor = ConsoleColor.Cyan; Console.WriteLine("****************************************************"); Console.WriteLine("* *"); Console.WriteLine("* Welcome to the Console Dashboard *"); Console.WriteLine("* *"); Console.WriteLine("****************************************************"); Console.ResetColor(); Console.WriteLine();

üí° Explanation:

  • ‚Ä¢ Console.ForegroundColor = changes text color.
  • ‚Ä¢ Console.WriteLine = prints text.
  • ‚Ä¢ Console.ResetColor() = returns to default text color.
  • ‚Ä¢ üé® Students can experiment with different colors if they want!

Ask the user for their name

Get user input

Console.ForegroundColor = ConsoleColor.DarkGreen; Console.WriteLine("Please enter your name:"); Console.ForegroundColor = ConsoleColor.White; userName = Console.ReadLine(); // waits for input and stores it in userName Console.ResetColor();

üí° Explanation:

  • ‚Ä¢ Console.ReadLine() pauses the program until the user types something.
  • ‚Ä¢ Whatever they type is stored in userName for later.

Create a loading effect

Add visual feedback

Console.Clear(); // clears the console Console.ForegroundColor = ConsoleColor.Red; Console.WriteLine("Loading the Dashboard"); for (int i = 0; i < 20; i++) { Console.Write("."); // print a dot Thread.Sleep(100); // pause 100ms between dots }

üí° Explanation:

  • ‚Ä¢ Makes the program feel interactive.
  • ‚Ä¢ Thread.Sleep(100) pauses for a short moment.
  • ‚Ä¢ Console.Clear() removes previous text for a clean screen.

Thread.Sleep(500); Console.Clear(); Console.ForegroundColor = ConsoleColor.DarkYellow; Console.WriteLine("************************"); Console.WriteLine("* ACCESS GRANTED *"); Console.WriteLine("************************"); Thread.Sleep(500); Console.WriteLine($"\nWelcome to the dashboard, {userName}!"); Console.ResetColor();

üí° Explanation:

  • ‚Ä¢ $"\nWelcome to the dashboard, {userName}!" ‚Üí $ allows variables inside text.
  • ‚Ä¢ Gives a personalized greeting to the user!

üí° Explanation:
